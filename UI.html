UI

Screens

For this section we will see each and every screen posible in Invoice Ninja, we will show an image or two of each screen and we will see different characteristics which define a good UI. Characteristics like:

  • Metaphors *
  • Color pallete
  • Font/Typography
  • Visual Aesthetics.
  • Graphic Components
  • Iconography
  • Patterns
  • Among other things including our own personal opinion

Let's get started


There are several different screens in the application, as it is exposed in the features section, the application itself has a lot of different features which each has one or more screens.

The application was developed having material design in mind. We can see this by the way the application is divided into card views and the way the buttons and popups interact and display. This will be explained further along with each view

Let's look at the first screen. The first screen that is shown is right when the application is loaded, corresponding to the logo screen:

This screen is simple enough, no text, no animations, nothing to interact with or to divert the view to. This screen has only the logo of invoice ninja.

Consisting of a dark collor pallete only composed of a few basic colors

Doing the inspection of the color pallete we can see that the color are a variation a degrees from black to white. Even more, as the image below shows, the color pallete is basically black and white, given that most of the diferent colors are located in the bar at the top.

This screen has no font other visual components or patters.

Nevertheless, it is a visually mesmerizing screeen because of the contratst between the black of the logo and the white of the background


Immediately after this screen, the user is placed on the dashboard screen. Which shows statisfics for invoices, payments, quotes, tasks & expenses here we show three of this elements, they are all organized one on top of another.

Looking only at this screen is very obvious that the application was developed having material design into mind. As we can see in the three images, they are all organized in a card like view, even more, we can see that some of these cards have a higher elevation than others, evident in the shadows that are displayed inbetween the date and the graphics
